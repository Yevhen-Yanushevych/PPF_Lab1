Laboratorium 1: git

Prykładowe użycie standartowych znaczników html

Praca z git:

  • Wykonaj i zapisz zmiany w kodzie projektu
  • Przejdż do panelu "Sorce Control"
  • Ustaw zmodyfikowanym plikom stan "Staged"
  • Wprowadz opis do pola tekstowego Message"
  • Zapisz zmiany lokalnie commit
  • Pobierz i skal zmiany zdalne pull
  • Rozwiąz ewentualne problemy
  • Wyslij aktualną wersje na server push
